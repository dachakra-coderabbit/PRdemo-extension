To complete the extension setup, add icon images in the following sizes:

- icon16.png (16x16 pixels) - Shown in extension management page
- icon48.png (48x48 pixels) - Shown in extension management page
- icon128.png (128x128 pixels) - Used in Chrome Web Store

You can:
1. Create simple icon images using any graphics editor
2. Use a robot/AI-themed icon to represent CodeRabbit
3. Use free icon resources like:
   - https://www.flaticon.com/
   - https://icons8.com/
   - https://www.iconfinder.com/

Temporary workaround:
- The extension will still work without icons
- Chrome will show a default placeholder icon
- Add proper icons before publishing

Recommended design:
- Use CodeRabbit's colors/theme if available
- Simple, recognizable design (robot or AI symbol)
- Looks good at small sizes (16x16)
